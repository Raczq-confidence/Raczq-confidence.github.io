JS当中的深浅拷贝以及数组扁平化

发布日期:   2020-12-24

1.深浅拷贝

因为JS的变量包含两种不同数据类型的值：基本类型和引用类型

1.JS当中的基本数据类型包含以下几种：

number，string，boolean，null，undefined，Symbol（ES6新增基本类型，之后会专门对他进行介绍）

2.JS当中的引用数据类型：

Object（Object，Array，Function）

两种数据类型最根本的区别在于两种变量的存储方式是不同的

JS存储变量的方式包含两种：栈内存（stack）和 堆内存（heap）

栈内存：自动分配内存空间，系统自动释放，里面存放的是基本类型的值和引用类型的地址

堆内存：动态分配的内存，大小不定，也不会自动释放。里面存放引用类型的值。

因为这样的存储方式，所以JS种两种数据类型的数据传递方式也不一样

  • 基本数据类型采用的是值传递

    let a = 10; // 定义一个变量a并赋值为10
let b = a;  // 将a的值10赋值给b (a、b都是基本类型，值传递)
b++;  // b自加
console.log(a, b) // 10, 11

  • 引用类型则是地址传递，将存放在栈内存中的地址赋值给接收的变量。

    let a = ['a', 'b', 'c']; // 定义一个数组a并赋值 
let b = a;   // 数组是引用类型,采用地址传递，将a的地址赋值给b
b.push('d'); // 给b数组增加一个'd'元素
console.log(a) // ['a', 'b', 'c', 'd']
console.log(b) // ['a', 'b', 'c', 'd']

深浅拷贝是指复制对象的级别深浅，具体来讲就是：

  • 浅拷贝是指复制对象的时候，只对第一层键值对进行独立的复制，如果对象内还有对象，则只能复制嵌套对象的地址
  • 深拷贝是指复制对象的时候完全的拷贝一份对象，即使嵌套了对象，两者也相互分离，修改一个对象的属性，也不会影响另一个。其实只要递归下去，把那些属性的值仍然是对象的再次进入对象内部，进行复制即可。

下面将为大家介绍深浅拷贝的几种实现方式

浅拷贝

1）Object.assign

let obj = {
    
    a : 10,
	b : [1,2,3],
	c : {
		x : 12
	},
	d : /^\d+$/ //正则表达式（也为引用数据类型）
    
}

let obj2 = Object.assign({},obj)

console.log(obj,obj2)

console.log('obj == obj2',obj == obj2) //false

console.log('obj === obj2',obj === obj2) //false

console.log('obj.__proto__ == obj2.__proto__',obj.__proto__ == obj2.__proto__) //true

obj2.c.x = 1000

console.log(obj.c.x)

2）Object.hasOwnProperty

let obj = {
	
	a : 10,
	b : [1,2,3],
	c : {
		x : 12
	},
	d : /^\d+$/ //正则表达式（也为引用数据类型）
	
}

let obj2 = {}

//浅拷贝
for(let key in obj){
	//判断属性是否为obj原型上的属性，是的话就结束循环
	if(!obj.hasOwnProperty(key)) break;
	obj2[key] = obj[key]
	
}

console.log(obj,obj2)

console.log('obj == obj2',obj == obj2) //false

console.log('obj === obj2',obj === obj2) //false

console.log('obj.__proto__ == obj2.__proto__',obj.__proto__ == obj2.__proto__) //true

obj2.c.x = 1000

console.log(obj.c.x)

因为浅拷贝的级别比较低，所以当我修改克隆后对象的属性（不为基本数据类型）时，原对象的属性也会被修改，原因就是引用数据类型会通过地址传递。

深拷贝

1）JSON.parse和JSON.stringify

let obj = {
	
	a : 10,
	b : [1,2,3],
	c : {
		x : 12
	}
 			
}

let obj2 = JSON.parse(JSON.stringify(obj))
console.log(obj2)

这种实现方法在项目中经常会使用到，缺点就是先转换为字符串再转换为对象，会开辟新的空间，并且当原对象中包含有函数（Function），日期（Date），正则表达式（RegExp）类型的属性时，就会出现下面例子出现的问题

let obj = {
	
	a : 10,
	b : [1,2,3],
	c : {
		x : 12
	},
	d : /^\d+$/,
	e : function () {
				
	}
 			
}

let obj2 = JSON.parse(JSON.stringify(obj))
console.log(obj2)

控制台输出的结果如下图所示

可以看出正则表达式类型的数据被转换为了{}，而函数类型的属性直接就消失不见了，所以我们可以采用下面的方式来进行深拷贝

2）递归转换

let obj = {
	
	a : 10,
	b : [1,2,3],
	c : {
		x : 12
	},
	d : /^\d+$/,
	e : function () {
				
	}
 			
}

function deepClone (obj) {
	//创建新的对象(通过这种方式创建的对象或者对象实例，他们的构造函数都是相同的)
	let newObj = new obj.constructor()
	
	//判断传入的实参是否为空
	if( obj === null ) return null
			
	//判断实参是否为对象(如果不为对象，则直接返回)
	if( typeof obj !== "object" ) return obj
	
	//判断传入实参是否为函数
	if( obj instanceof Function ){
				
		//创建返回值为新的Function实例
		return new Function(obj)
				
	}
			
	//判断传入实参是否为正则表达式
	if( obj instanceof RegExp ){
				
		//创建返回值为新的正则表达式实例
		return new RegExp(obj)
				
	}
			
	//判断传入实参是否为Date
	if( obj instanceof Date ){
				
		//创建返回值为新的Function实例
		return new Date(obj)
				
	}
			
	//
	for( let key in obj ){
				
		//需要对数据进行过滤
		newObj[key] = deepClone(obj[key])
				
	}
			
	return newObj
			
}
		
let obj2 = deepClone(obj)
		
console.log(obj,obj2)
		
console.log('obj2 === obj',obj2 === obj) //false
		
console.log('obj2.c === obj.c',obj2.c === obj.c) //false

可以看到成功对obj对象进行了深拷贝

2.数组扁平化

数组的扁平化，就是将一个嵌套多层的数组 array (嵌套可以是任何层数)转换为只有一层的数组。

举个例子，假设有个名为 flatten 的函数可以做到数组扁平化，效果就会如下：

var arr = [1, [2, [3, 4]]]
console.log(flatten(arr)) // [1, 2, 3, 4]

接下来我们将去尝试实现数组扁平化（下文数组扁平化的目标都将为下面的数组）

var arr = [
	[1],
	[2],
	[3,4,[5,[6,[7]]]],
	8
] //这是一个“四层”数组

flat方法实现

flat方法是ES6新提出的方法

flat方法会按照一个可指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回。他挂载在Array.prototype上

他需要传入一个数字，指定要提取嵌套数组的结构深度，默认值为 1，当传入参数为 3 时

arr = arr.flat(3)
console.log(arr)

结果如下图所示

当传入参数为 Infinity 时，Infinity 便是遍历无穷层数，将其转为“一维”数组

arr = arr.flat(Infinity)
console.log(arr)

结果如下图所示

toString方法

将需要扁平化的数组先转换为字符串，再转换为数组，之后将每一项变为数字，实现方法如下

arr = arr.toString().split(',').map( item => Number(item) )
console.log(arr)

结果如下图所示

利用正则表达式

arr = JSON.stringify(arr).replace(/(\[|\])/g,'').split(',').map( item => Number(item) )
console.log(arr)

结果如下图所示

循环验证

循环验证元素是否为数组(判断条件中的回调函数也可以写Array.isArray(item))

while( arr.some( item => item instanceof Array ) ){
	
	arr = [].concat(...arr)
			
}
